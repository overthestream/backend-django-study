웹은 기본적으로
  클라이언트 - 서버 간의 상호작용 

  HTTP 프로토콜 : Request <-> Response 

컴퓨터 네트워크 -> 인터넷 -> 웹(월드 와이드 웹)
컴퓨터 네트워크 
  분산되어 있는 컴퓨터들을 통신망으로 연결한 것 
  케이블 매체가 무선 매체를 통해 서로에게 데이터를 교환
인터넷 
  TCP/IP를 이용해 정보를 주고받는 컴퓨터 네트워크 
웹 
  웹 자원의 위치 URL + 웹 자원의 이름에 접근하는 HTTP + 자원들 사이를 쉽게 항해하는 HTML
  
하이퍼텍스트 : 참조(하이퍼링크)를통해 독자가 한 문서에서 다른 문서로 즉시 접근할 수 있는 텍스트 

HTTP: HyperText Transfer Protocol
  -> HTML 문서를 주고 받는데 주로 쓰임 
  -> 클라이언트와 서버 사의 요청/응답 프로토콜 
  -> TCP / IP위에서 동작함 
    TCP / IP
      Transmission Control Protocol 
        IP의 핵심 프로토콜
  -> 클라이언트 (주로 웹 브라우저)가 HTTP로 서버에 
    HTML이나 이미지 등을 Request 하면 서버는 그 Response를 클라이언트에 전달
  -> URL을 사용해 조회 가능
    Uniform Resource Locator, 웹 주소
    네트워크 상에서 자원이 어디 있는지를 알려주는 규약
